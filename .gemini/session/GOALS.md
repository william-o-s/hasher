# Stretch Goals

This document tracks stretch goals for the File Hasher Desktop application, to be implemented after the core functionality is achieved.

## General
- [ ] Add Drag and Drop support for files (dropping over the window sets the selected file).
- [ ] Add Drag and Drop support for text files or `.sha256` files directly into the expected hash input.

## Features
- [ ] Add support for adding alternative hashing algorithms (MD5, SHA1, SHA512) via a builder interface where the user can select multiple algorithms to hash the file with. Each algorithm should run in parallel.
- [ ] Improve dark/light mode thematic switches if user OS settings dictate.
